# 🏡 EstateNexa

**EstateNexa** is a Smart Real Estate Advisory Assistant powered by Agentic AI that helps users discover properties, compare options, and make intelligent investment decisions.

---

## 🚀 Overview

EstateNexa leverages AI-driven insights to simplify real estate decisions by:

* Retrieving property data
* Providing personalized recommendations
* Analyzing market trends
* Maintaining user preferences
* Supporting role-based interactions

---

## 🧠 Key Features

* 🔍 **Property Search & Retrieval**
* 🤖 **AI-Powered Recommendations**
* 📊 **Market Trend Analysis**
* 🧾 **Property Comparison**
* 💡 **Investment Suggestions**
* 👤 **User Preference Memory**
* 🔐 **Role-Based Access (Admin / Agent / Buyer)**

---

## 🏗️ Architecture

### 1. Presentation Layer

* Streamlit UI
* Chat Interface
* Login System

### 2. Application Layer

* FastAPI Backend
* API Handling

### 3. Intelligence Layer

* LangChain Agent
* Real Estate Reasoning Engine

### 4. Data Layer

* Vector Database (Property listings, reports)
* PostgreSQL (Users, preferences, memory)

### 5. Observability Layer

* LangSmith (Tracking & Monitoring)
